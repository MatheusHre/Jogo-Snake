# 🎮 Jogo da Cobra em Python com Pygame

Este projeto foi desenvolvido como parte das atividades da faculdade, com o objetivo de aplicar na prática conceitos de programação utilizando Python e a biblioteca Pygame.

## 🎯 Sobre o projeto

O projeto consiste em um jogo simples desenvolvido em Python, onde o jogador controla um personagem e deve evitar obstáculos para sobreviver o maior tempo possível.

## 🕹️ Como jogar

* Utilize as teclas do teclado para movimentar o personagem
* Desvie dos obstáculos que aparecem na tela
* O objetivo é sobreviver o máximo de tempo possível
* Ao colidir com um obstáculo, o jogo exibe a tela de **Game Over**
* Pressione uma tecla para reiniciar o jogo
